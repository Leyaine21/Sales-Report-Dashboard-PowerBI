# Sales-Report-Dashboard-PowerBI
## Overview 
This project is an interactive Sales Report Dashboard developed using Microsoft Power BI. The dashboard provides a visual overview of sales performance across different categories, regions, and time periods, enabling users to identify trends and monitor key business metrics. As my first Power BI project, this dashboard helped me gain practical experience in data modeling, data visualization, report design, and business intelligence reporting.

## Project Objectives 
 - Analyze sales performance across multiple categories.
 - Track monthly sales and quantity trends.
 - Compare sales performance by business category.
 - Visualize geographic sales distribution.
 - Build an interactive dashboard using Power BI.
   
## Dashboard Features
### Year Filter
Users can filter dashboard data by year:
- 2018
- 2019
- 2020
- 2021

### Key Performance Indicator (KPI)
Total Quantity Sold: 9 Million

### Sales by Category
The dashboard compares sales across customer/business categories:
- Novelty Shop
- Supermarket
- Gift Store
- Computer Store
- Corporate

### Monthly Sales Analysis
A sales table displays:
- Monthly sales values
- Year-over-Year (YoY%) performance
- Comparative sales metrics

### Quantity Trend Analysis
A line chart visualizes quantity sold by month, making it easy to identify seasonal trends and performance fluctuations.

### Geographic Analysis
A map visualization displays sales distribution across provinces/regions, providing location-based business insights.
